# General rules

## Do

- Use `text-brand-navy` for headings.
- Use `text-brand-muted` for supporting text.
- Use `bg-brand-primary text-white` for primary CTAs.
- Use `bg-brand-soft` for soft page sections.
- Use `bg-white border-brand-border` for cards.
- Use `bg-brand-blueSoft text-brand-primary` for secondary actions.
- Use `text-brand-secondary` only as an accent.
- Use generous spacing.
- Prefer rounded corners.
- Keep shadows subtle.

## Do not

- Do not hardcode hex colors inside React components.
- Do not use random Tailwind colors like `blue-600`, `purple-500`, or `slate-700` when a brand token exists.
- Do not use purple as the main CTA color.
- Do not overuse green.
- Do not use heavy shadows.
- Do not make every section dark.
- Do not create high-saturation gradients unless specifically requested.
- Do not mix many unrelated accent colors.
